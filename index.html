Welcome to PAB

What is PAB?

PAB (a very original name - Project Access Bot ) is a chat bot to get you aware for what requirements you have when it comes to the university of your dreams. This project is meant to be an extension of the existing effort made by Project Access to improve accessibility for students to top universities. Find more about them by clicking on the button bellow.
